3. Turtle Graphics

    First, you need to load Turtle Graphics:
    
    `
    import turtle
    `
    
    Then, you can draw pictures:
    
    `
    turtle.forward(100)
    turtle.left(90)
    turtle.forward(200)
    `

    You can change the color:
    
    `
    turtle.pencolor("red")
    `

    You can change the background color:
    
    `
    turtle.bgcolor("blue")
    `
