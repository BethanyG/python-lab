3. Turtle Graphics

    First, you need to load Turtle Graphics:
    
    ```python
    import turtle
    ```
    
    Then, you can draw pictures:
    
    ```python
    turtle.forward(100)
    turtle.left(90)
    turtle.forward(200)
    ```

    You can change the color:
    
    ```python
    turtle.pencolor("red")
    ```

    You can change the background color:
    
    ```python
    turtle.bgcolor("blue")
    ```
